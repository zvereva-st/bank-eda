# Exploratory Data Analysis: Bank Marketing Dataset

## Описание проекта
Данный проект представляет собой разведочный анализ данных (EDA) на основе набора данных Bank Marketing.  
Цель исследования — изучить характеристики клиентов и выявить факторы, влияющие на вероятность отклика (`y`) на маркетинговую кампанию банка.

Анализ выполнен в Google Colab и включает обзор структуры данных, визуализацию распределений, проверку качества данных, расчет корреляций и сегментный анализ.

## Используемые данные
Dataset: Bank Marketing — информация о результатах маркетинговых звонков клиентам банка.  
Целевая переменная: `y` — факт подписки клиента на депозит (yes/no).  
Формат: CSV, разделитель `;`.

Основные группы признаков:
- Социально-демографические характеристики (`age`, `job`, `education` и др.)
- Финансовая информация (`housing`, `loan`, `default`)
- Маркетинговые параметры (`contact`, `month`, `campaign`, `duration`)
- История предыдущих кампаний (`pdays`, `poutcome`)

## Выполненные этапы анализа

### 1. Первичный обзор данных
- Загрузка данных и проверка размера датасета  
- Просмотр первых строк  
- Анализ распределения целевой переменной  
- Рассмотрение основных статистических характеристик  

### 2. Проверка качества данных
- Определение типов признаков  
- Проверка пропущенных значений  
- Поиск и анализ дубликатов  

### 3. Анализ числовых признаков
- Гистограммы распределений  
- Выявление выбросов и особенностей распределений  

### 4. Анализ категориальных признаков
- Оценка доли откликов `yes` внутри групп  
- Визуализация важнейших категориальных сегментов  

### 5. Корреляционный анализ
- One-Hot Encoding для категориальных переменных  
- Расчет корреляции признаков с целевой переменной  
- Выделение наиболее связанных переменных  

### 6. Сегментный анализ
Исследованы сегменты по признакам:
- `job`
- `education`
- `contact`
- `month`
- `poutcome`

Сегментация позволяет увидеть, какие категории клиентов реагируют лучше всего

### 7. Выводы
- Расчитана общая конверсия по датасету  
- Определены признаки с наибольшей корреляцией с откликом  
- Сформированы лидирующие сегменты для дальнейшего анализа

## Итоговые выводы по EDA

### 1. Общая конверсия низкая
Средняя конверсия по датасету составляет 11.27%.  
Это характерно для холодных маркетинговых кампаний и создает задачу поиска наиболее перспективных сегментов.

### 2. Сильные различия между сегментами
EDA показывает, что вероятность отклика сильно зависит от характеристик клиента и особенностей кампании.

Наиболее заметные наблюдения:
- Клиенты, с которыми связывались через `cellular`, показывают более высокую конверсию по сравнению с `telephone`.
- Месяцы проведения кампании сильно различаются: лучшие результаты наблюдаются в марте, сентябре, декабре.
- Статус предыдущей кампании (`poutcome`) является важным фактором: клиенты, у которых ранее результат был "success", имеют существенно более высокую вероятность отклика.
- Отдельные профессии (например, студенты и пенсионеры) демонстрируют повышенный интерес по сравнению с другими группами.

### 3. Признак `duration` обладает самой высокой корреляцией
Однако он не подходит для прогнозирования, так как фиксируется только после звонка.

### 4. Числовые признаки имеют выраженную асимметрию
Ряд признаков требует трансформации для дальнейшего моделирования.

### 5. Категориальные признаки — ключевые драйверы результата
Особенно информативны:
- `contact`
- `month`
- `poutcome`
- `education`
- `job`

### 6. Датасет сбалансирован по качеству
- Отсутствуют пропуски  
- Количество дубликатов невелико  
- Признаки хорошо структурированы
